mark·up / ˈmɑːkʌp ; NAmE ˈmɑːrkkʌp / noun 1 [usually singular ] an increase in the price of sth based on the difference between the cost of producing it and the price it is sold at （基于成本价与销售价之间差价的）加成 ◆ an average markup of 10% 平均 10% 的加成 2 [uncountable ] ( computing 计 ) the symbols used in computer documents which give information about the structure of the document and tell the computer how it is to appear on the computer screen, or how it is to appear when printed 标记，标记符号（标示计算机文档结构和显示、打印形式） ◆ a markup language 标记语言 mark·up / ˈmɑːkʌp ; NAmE ˈmɑːrkkʌp /
