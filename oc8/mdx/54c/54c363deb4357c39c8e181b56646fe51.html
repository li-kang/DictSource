at·tribu·tive / əˈtrɪbjətɪv ; NAmE əˈtrɪbjətɪv / adjective ( grammar 语法 ) (of adjectives or nouns 形容词或名词 ) used before a noun to describe it （用于所修饰的名词前）定语的 ◆ In ‘the blue sky’ and ‘a family business’, ‘blue’ and ‘family’ are attributive. 在 the blue sky 和 a family business 中，blue 和 family 是定语。 ☞compare predicative ▶ at·tribu·tive·ly / əˈtrɪbjətɪvli ; NAmE əˈtrɪbjətɪvli / adverb ◆ Some adjectives can only be used attributively. 有些形容词只能用作定语。 at·tribu·tive / əˈtrɪbjətɪv ; NAmE əˈtrɪbjətɪv / at·tribu·tive·ly / əˈtrɪbjətɪvli ; NAmE əˈtrɪbjətɪvli /
