for·bid / fəˈbɪd ; NAmE fərˈbɪd / verb ( for·bade / fəˈbæd ; NAmE fərˈbæd / / fəˈbeɪd ; NAmE fərˈbeɪd / for·bid·den / fəˈbɪdn ; NAmE fərˈbɪdn / ) 1 to order sb not to do sth; to order that sth must not be done 禁止；不准 forbidsb (from doing sth) ◆ He forbade them from mentioning the subject again. 他不准他们再提到这个问题。 forbidsth ◆ Her father forbade the marriage. 她的父亲不允许这桩婚事。 forbidsb to do sth ◆ You are all forbidden to leave. 你们都不准离开。 forbidsb sth ◆ My doctor has forbidden me sugar. 医生禁止我吃糖。 forbid(sb) doing sth ◆ She knew her mother would forbid her going. 她知道她妈妈是不会让她去的。 OPP allow , permit 2 forbidsth | forbidsb to do sth ( formal) to make it difficult or impossible to do sth 妨碍；阻碍；阻止 SYN prohibit ◆ Lack of space forbids further treatment of the topic here. 由于篇幅所限，这里不能深入阐述这个问题。 IDIOM God/Heaven forˈbid (that…) ( informal) used to say that you hope that sth will not happen 但愿这事不发生 ◆ ‘Maybe you'll end up as a lawyer, like me.’ ‘God forbid!’ “也许你会像我一样，最终成为律师。”“但愿不会这样！” HELP  Some people find this use offensive. 有人认为此用法含冒犯意。 forbid forbids forbade forbidding forbidden for·bid / fəˈbɪd ; NAmE fərˈbɪd / for·bade / fəˈbæd ; NAmE fərˈbæd / for·bid·den / fəˈbɪdn ; NAmE fərˈbɪdn /
