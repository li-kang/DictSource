hang·man / ˈhæŋmən ; NAmE ˈhæŋmən / noun ( plural hang·men / ˈhæŋmən ; NAmE ˈhæŋmən / ) 1 [countable ] a man whose job is to hang criminals 执行绞刑的人；刽子手 2 [uncountable ] a game in which one player chooses a word and the other players try to guess it, letter by letter. Each time they guess wrongly, the first person draws one part of a man being hanged. The other players have to guess the word before the drawing is complete. 猜词画人游戏（一个玩家选一个单词，其他玩家设法猜出，每猜错一个字母就画一部份吊着的人的身体，其他玩家须在全部身体画完之前猜出该词。） hangman hangmen hang·man / ˈhæŋmən ; NAmE ˈhæŋmən / hang·men / ˈhæŋmən ; NAmE ˈhæŋmən /
