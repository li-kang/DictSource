ka-ching / ; NAmE / ( BrE also ker-ching ) ( NAmE also cha-ching ) exclamation ( informal) used to say that sb is getting a lot of money 哗啦（表示赚大钱） ◆ The money was rolling in, ka-ching, ka-ching! 财富哗啦啦地滚滚而来！ ◆ Ka-ching! I just got my first cheque. 哗啦！我刚得到第一张支票。 ORIGIN A way of representing the noise made by a cash register. 源自对收银机声响的模仿。 ka-ching / ; NAmE /
