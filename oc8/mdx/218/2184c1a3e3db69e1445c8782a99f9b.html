gotcha / ˈɡɒtʃə ; NAmE ˈɡɑːtʃə / exclamation ( non-standard) the written form of the way some people pronounce ‘I've got you’, which is not considered to be correct （对 I've got you 发音的书写形式，此用法被视为不正确） ◆ ‘Gotcha!’ I yelled as I grabbed him by the arm. (= used when you have caught sb, or have beaten them at sth). “逮住你了！”我抓住他的胳膊大声喊道。 ◆ ‘Don't let go.’ ‘Yeah, gotcha.’ (= Yes, I understand.) “别松手。”“是，我明白。” HELP  You should not write this form unless you are copying somebody's speech. 这种写法只用于引述别人的话。 gotcha / ˈɡɒtʃə ; NAmE ˈɡɑːtʃə /
