olde / əʊld ; NAmE oʊld / / ˈəʊldi ; NAmE ˈoʊldi / adjective [only before noun ] ( old use) a way of spelling ‘old’ that was used in the past and is now sometimes used in names and advertisements to give the impression that sth is traditional （old 的旧式拼法，现有时用于名称或广告，以使人感到某物是传统的） ◆ a pub that tries to recreate the flavour of olde England 一家力图重现古英国风情的酒馆 olde / əʊld ; NAmE oʊld /
