Sisy·phean / ˌsɪsɪˈfiːən ; NAmE ˌsɪsɪˈfiːən / adjective (of a task 任务 ) impossible to complete 不可能完成的；永无休止的 ORIGIN From the Greek myth in which Sisyphuswas punished for the bad things he had done in his life with the never-ending task of rolling a large stone to the top of a hill, from which it always rolled down again. 源自希腊神话，西绪福斯（Sisyphus）因前生罪恶受惩罚，无休止地将一块巨石磙到山顶，而巨石总是一再滚落。 Sisy·phean / ˌsɪsɪˈfiːən ; NAmE ˌsɪsɪˈfiːən /
