place·hold·er / ˈpleɪshəʊldə(r) ; NAmE ˈpleɪshoʊldər / noun 1 ( technical 术语 ) a symbol or piece of text which replaces sth that is missing （替代缺失部份的）占位符，占位文字 2 ( linguistics 语言 ) an item which is necessary in a sentence, but does not have real meaning, for example the word ‘it’ in ‘It's a pity she left.’ 位标（句子中必要但无实际意义的词项，如 It's a pity she left 中的 it） place·hold·er / ˈpleɪshəʊldə(r) ; NAmE ˈpleɪshoʊldər /
