syn·ec·doche / sɪˈnekdəki ; NAmE sɪˈnekdəki / noun [uncountable ,  countable ] ( technical 术语 ) a word or phrase in which a part of sth is used to represent a whole, or a whole is used to represent a part of sth. For example, in ‘Australia lost by two goals’, Australiais used to represent the Australian team. 举偶法，提喻法（用局部代表整体或用整体代表局部的修辞手段） synecdoche synecdoches syn·ec·doche / sɪˈnekdəki ; NAmE sɪˈnekdəki /
