ana·logy AWL / əˈnælədʒi ; NAmE əˈnælədʒi / noun ( plural ana·logies ) 1 [countable ] a comparison of one thing with another thing that has similar features; a feature that is similar 类比；比拟；相似之处 analogy(between A and B) ◆ The teacher drew an analogybetween the human heart and a pump. 老师打了个比喻，把人的心脏比作水泵。 analogy(with sth) ◆ There are no analogies with any previous legal cases. 以往的法律案件没有哪一宗可与此案类比。 2 [uncountable ] the process of comparing one thing with another thing that has similar features in order to explain it 类推；比拟 ◆ learning by analogy 用类推法学习 analogy analogies ana·logy / əˈnælədʒi ; NAmE əˈnælədʒi /
