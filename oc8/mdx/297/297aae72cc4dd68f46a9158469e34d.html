crazy ★ / ˈkreɪzi ; NAmE ˈkreɪzi / adjective , noun crazy crazier craziest adjective ★ ( cra·zier , crazi·est ) ( informal) 1 ★ ( especially NAmE) not sensible; stupid 不理智的；疯狂的；愚蠢的；傻的 ◆ Are you crazy? We could get killed doing that. 你疯了？我们那样做会丧命的。 ◆ She must be crazy to lend him money. 她把钱借给他，一定是疯了。 ◆ He drove like an idiot, passing in the craziest places. 他像白痴一样净把车往最不可思议的地方开。 ◆ What a crazy idea! 这个想法真荒唐！ ◆ I know it sounds crazy but it just might work. 我知道这事听起来很蠢，然而也许就会奏效。 2 ★ very angry 非常气愤 ◆ That noise is driving me crazy. 那噪声吵得我快发疯了。 ◆ Marie says he went crazy,and smashed the room up. 玛丽说他气得发疯，把房间里的东西都砸碎了。 3 ★ crazy(about sth) ( often in compounds 常构成复合词 ) very enthusiastic or excited about sth 热衷的；狂热的 ◆ Rick is crazy about football. 里克对足球着了迷。 ◆ He's football-crazy. 他是个足球迷。 ◆ I'm not crazy about Chinese food (= I don't like it very much). 我并不十分热衷于中国菜。 ◆ The crowd went crazywhen the band came on stage. 乐队出场时，观众欣喜若狂。 ◆ You're so beautiful you're driving me crazy. 你真美，弄得我神魂颠倒了。 4 ★ crazyabout sb liking sb very much; in love with sb 迷恋的；爱上的 ◆ I've been crazy about him since the first time I saw him. 我从第一次见面就爱上他了。 5 ★ ( especially NAmE) mentally ill; insane 患精神病的；精神失常（或错乱）的 ◆ She's crazy—she ought to be locked up. 她疯了，应该把她关起来。 ☞synonyms at mad ▶ crazi·ly / ; NAmE / adverb cra·zi·ness / ˈkreɪzinəs ; NAmE ˈkreɪzinəs / noun [uncountable ] IDIOM like ˈcrazy/ˈmad ( informal) very fast, hard, much, etc. 非常快地；拚命地；疯狂地 ◆ We worked like crazy to get it done on time. 我们拚命地干，好按时完成这项工作。 noun ( plural crazies ) ( informal) ( especially NAmE) a crazy person 疯子 crazy / ˈkreɪzi ; NAmE ˈkreɪzi / crazi·ly / ; NAmE / cra·zi·ness / ˈkreɪzinəs ; NAmE ˈkreɪzinəs /
