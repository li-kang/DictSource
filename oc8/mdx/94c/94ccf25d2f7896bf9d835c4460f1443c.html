schlep ( also schlepp ) / ʃlep ; NAmE ʃlep / verb ( -pp- ) ( informal) ( especially NAmE) 1 [intransitive ] + adv./prep. to go somewhere, especially if it is a slow, difficult journey, or you do not want to go （劳神耗时或不情愿地）去，赶往 2 [transitive ] schlepsth (+ adv./prep.) to carry or pull sth heavy 搬，抬，拖，拉（重物） ◆ I'm not schlepping these suitcases all over town. 我可不想提着这几个行李箱满城跑。 ORIGIN From Yiddish shlepn,‘to drag’. 源自意第绪语 shlepn，意为“拖”。 ▶ schlep ( also schlepp ) noun [singular ] schlep schleps schlepped schlepping schlep / ʃlep ; NAmE ʃlep /
