hop·scotch / ˈhɒpskɒtʃ ; NAmE ˈhɑːpskɑːtʃ / noun [uncountable ] a children's game played on a pattern of squares marked on the ground. Each child throws a stone into a square then hops(= jumps on one leg) and jumps along the empty squares to pick up the stone again. 跳房子（儿童单足跳石子的游戏） hop·scotch / ˈhɒpskɒtʃ ; NAmE ˈhɑːpskɑːtʃ /
