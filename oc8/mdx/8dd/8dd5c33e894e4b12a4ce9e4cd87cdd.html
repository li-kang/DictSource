clas·si·fier / ˈklæsɪfaɪə(r) ; NAmE ˈklæsɪfaɪər / noun ( grammar 语法 ) an affixor word which shows that a word belongs to a group of words with similar meanings. For example the prefix ‘un’ is a classifierthat shows the word is negative. 分类成分，分类词（能显示同义所属关系，如前缀 un 是表示反义的分类成分） classifier classifiers clas·si·fier / ˈklæsɪfaɪə(r) ; NAmE ˈklæsɪfaɪər /
